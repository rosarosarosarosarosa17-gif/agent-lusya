# Agent Lusya

## Что это

Скелет «взрослого» Node.js-агента с разделением на этапы жизни:
`init` (запуск), `tick` (одна итерация работы) и graceful `shutdown`
(аккуратная остановка по сигналу). Пока это каркас — начинку (память,
мозг-LLM, Telegram) добавим в Разделе 3.

## Как запустить

```bash
cp .env.example .env
npm install
npm start
```

## Переменные окружения

| Переменная   | Назначение                         | По умолчанию |
|--------------|------------------------------------|--------------|
| `AGENT_NAME` | Имя агента                         | `Agent Lusya`|
| `TICK_MS`    | Период тика в миллисекундах        | `10000`      |

## Деплой

Деплой автоматический: `push` в ветку `main` запускает GitHub Action,
который заливает проект на VPS по SCP.

Для работы нужны секреты репозитория (Settings → Secrets and variables →
Actions):

- `VPS_HOST` — адрес сервера
- `VPS_USER` — пользователь
- `VPS_PASSWORD` — пароль

Файлы копируются в `/var/www/agent-lusya/` (без `node_modules`, `.git`, `.env`).
